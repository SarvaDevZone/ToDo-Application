# Todo Application

A simple and elegant Todo application for managing your tasks effectively.

---

## Features
- Add tasks
- Edit tasks inline
- Delete tasks
- Responsive design

---

## Demo

---

## How to Use
1. **Add Tasks**:
   - Enter your task in the input field.
   - Click "Add Task" or press `Enter`.

2. **Edit Tasks**:
   - Click the ✏️ icon next to a task.
   - Modify the task and save your changes.

3. **Delete Tasks**:
   - Click the 🗑️ icon to remove a task.
